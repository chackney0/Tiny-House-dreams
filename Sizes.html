Big Tiny

A big tiny is a tiny house that is still over 250 square feet. However it is under 400 square feet.This house would be able to fit multipule elements into it. You could choose to have full bathroom. By tiny house standards this is a big house. It still has the option to be mobile however its more likely going to be put in a permenant location.

Small Tiny

A small tiny ranges from 15-250 square feet. In this size range is considerably small how ever there is still room to put in all the elements that you would want. This size is more desireable if you would like to have the option to still have a lot of features while still being able to be mobile if you choose.

Teeny Tiny

A teeny tiny is under 150 feet. At this size it becomes much more of a challenge to get all the features you may want. How ever it is completely do able and liveable. This house is another great choice if you want to be mobile. Like all these choices its still a great option. Its all about choosing which one fits you best.
